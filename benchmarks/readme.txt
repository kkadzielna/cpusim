Programs can be executed by running the command "python3 main.py filename mode", where "filename" denotes the name of the file
with the assembly code to be executed by the CPU simulation, while "mode" signifies the mode the CPU will be used in. The available modes
are "simple", "pipelined" and "superscalar". The provided programs are "bubblesort.txt", "fibonacci.txt", "raw.txt" and "indepenent_artithmetic.txt". 
The input to those programs is currently hardcoded and cannot be provided as an argument.
